### 🤖 Agentic AI & LLMs Focus Areas
* **Multi-Agent Orchestration:** Designing collaborative agent frameworks where multiple specialized agents communicate, assign tasks, and solve complex problems (using **CrewAI**, **LangGraph**, **AutoGen**).
* **Model Context Protocol (MCP):** Implementing custom MCP servers and clients to connect LLMs with local/remote tools, databases, and development environments.
* **Retrieval-Augmented Generation (RAG):** Building production-ready, high-accuracy RAG pipelines with advanced retrieval techniques (hybrid search, parent-document retrieval).
* **Human-in-the-Loop (HITL) & Guardrails:** Engineering safe agentic workflows with strict validation guardrails (NeMo Guardrails, Llama Guard) and interactive human verification steps.

### 📁 Featured Projects (Agentic AI & RAG Focus)

* **🤖 Multi-Agent MCP Collaboration System**
  * Built a system of cooperative agents communicating via the **Model Context Protocol (MCP)**, allowing agents to execute complex filesystem operations, database queries, and tool utilization in a secure environment.
  * *Tech stack: Python, MCP SDK, LangGraph, SQLite, Docker.*

* **🛡️ HITL-Enabled Guardrails & Responsible AI Agent**
  * Developed a responsible AI agent with integrated human-in-the-loop (HITL) approval mechanisms and policy-based guardrails (checking semantic similarity, PII scanning, and toxicity detection) before executing high-risk database transactions.
  * *Tech stack: Python, LangChain, NeMo Guardrails, PostgreSQL.*

* **⚡ High-Performance RAG Pipeline**
  * Built a cohort-focused RAG system utilizing hybrid keyword-dense retrieval, semantic embeddings, parent-document retrievers, and Hugging Face rerankers to output hallucination-free summaries of academic lectures.
  * *Tech stack: PyTorch, FAISS, LlamaIndex, Python, Hugging Face.*
