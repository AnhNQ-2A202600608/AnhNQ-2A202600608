### 🧠 Areas of Interest
* **Multi-Agent Orchestration:** Studying collaborative agent frameworks where multiple specialized agents communicate, distribute sub-tasks, and solve problems (exploring **CrewAI**, **LangGraph**, and **AutoGen**).

* **Reliable RAG Architectures:** Experimenting with hybrid search, query rewriting, and rerankers to improve factual consistency and reduce hallucinations in retrieval pipelines.

  * Exploring cooperative agent communication via the **Model Context Protocol (MCP)**, allowing agents to coordinate filesystem tasks, run local database queries, and manage simple tool utilization.
  * *Tech stack: Python, MCP SDK, LangGraph, SQLite, Docker.*

* **🛡️ HITL-Enabled Guardrails & Responsible AI Agent**
